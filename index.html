Información del proyecto

SGD

Introducción

El software se ajustará a las necesidades del área de gestión humana y se podrá realizar los ajustes solicitados por las directivas de cada empresa, también será actualizado por el área de gestión humana cuando se realicen cambios o ingresen más documentos a las historias laborales o se registren más historias laborales y de esta manera se podrán visualizar o descargar reportes de cada empleado según lo que indique el área que requiera los reportes.

SGD

Objetivos

Objetivo general

  • Gestionar el control de las historias laborales en el área de recursos humanos

Objetivos especificos

  • Disminuir el uso de papel en el área de recursos humanos
  • Controlar el ciclo de vida de las historias laborales
  • Incrementar la seguridad, agilidad y recuperación de la información de las historias laborales

SGD

Alcance

Este documento describe los requerimientos de software del Desarrollo de un sistema de gestión documental para el control de historias laborales en el área de gestión humana, cuyo objetivo principal es Gestionar el control de las historias laborales en el área de recursos humanos Nuestro sistema de información se desarrollará específicamente para el área de talento humano, para el cual pasaremos por las siguientes fases:

  • Levantamiento de requerimientos.
  • Diseño del Software (UML).
  • Desarrollo del producto.
